#Sum of Integers in a Range
p:Given a range [m, n] (both inclusive) where 0 <= m, n <= 10000, find the sum of all integers between m and n.

Example:
Input:
0 3
Output:
6
Explanation:
0+1+2+3 = 6

CODING:
m=int(input())
n=int(input())
result = sum(range(m, n + 1))           
print(result)
